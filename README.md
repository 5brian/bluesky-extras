# bluesky-extras

chrome extension to show DID tools in bsky.app 
 
## dev

1. clone:

   ```
   git clone https://github.com/yoleuh/bluesky-extras.git
   ```

2. run:

   ```
   tsc
   ```

3. nav:

   ```
   chrome://extensions/
   ```

4. open `Load unpacked`

5. select `chrome`

## usage

1. open profile

2. select tool from footer
